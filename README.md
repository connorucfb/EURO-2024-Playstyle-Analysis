# EURO 2024 Team Playstyle Analysis

This project investigates team playing styles at UEFA EURO 2024 using StatsBomb event and 360 data. 

The analysis combines Python and statistical techniques to identify tactical similarities and differences between international teams.

## Objective

Develop an automated Python workflow to analyse StatsBomb event and 360 data, identify team playstyles and output visualisations.

## Repository Structure

- **EURO24_Analysis.ipynb** – Main analysis notebook containing the complete data preparation, analysis and visualisations.
- **Research Project.pdf** – Final research report describing the methodology, results and conclusions.

## Technologies

- Python
- pandas
- NumPy
- scikit-learn
- StatsBombPy
- mplsoccer
- Matplotlib
